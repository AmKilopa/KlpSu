## Интерфейс

### 1. Главный экран

  <img src="https://github.com/user-attachments/assets/91c9c3cd-d1dd-44d4-96f6-e61c4849a2fa" alt="Главный экран KlpSu" width="900" />
</p>

На главном экране находится основной блок работы со ссылкой:

- **Поле «Длинная ссылка»** — сюда вставляется исходный URL, который нужно сократить.  
- **Кнопка «Дополнительные настройки»** разворачивает блок с продвинутыми параметрами (кастомный код, срок действия, лимит кликов, пароль).  
- **Кнопка «Сократить ссылку»** запускает процесс: проверяется валидность ссылки, ищутся существующие варианты или готовится превью новой короткой ссылки.

### 2. Дополнительные настройки

  <img width="1100" height="800" alt="image" src="https://github.com/user-attachments/assets/ac2141ce-2922-40fa-b748-d0367b92fabd" />

В блоке дополнительных настроек можно тонко настроить поведение короткой ссылки:

- **Кастомный код** — позволяет задать свой короткий код (ровно 6 символов). Если поле пустое, код будет сгенерирован автоматически.  
- **Срок действия** — определяет, до какой даты и времени ссылка будет считаться активной. После истечения срока пользователь увидит страницу «Срок истек».  
- **Макс. кликов** — ограничивает количество переходов по ссылке; при достижении лимита покажется страница «Лимит исчерпан».  
- **Переключатель «Защитить паролем»** включает защиту: при открытии короткого URL сначала показывается форма ввода пароля, и только после успешной проверки выполняется переход на оригинальный адрес.  
- **Поле «Введите пароль»** — задаёт пароль, который нужно будет ввести перед переходом по ссылке.

  <img width="1100" height="700" alt="image" src="https://github.com/user-attachments/assets/0846f673-7f22-4b06-8078-21aebd6dfaab" />

Если для одного длинного URL уже создавались короткие ссылки, они отображаются в отдельном блоке:

- Строка с **коротким URL** показывает ссылку вида `https://.../код`, по которой можно перейти.  
- Под ссылкой отображаются **счётчики**: количество использованных кликов, установленный лимит и оставшееся время до истечения срока действия.  
- **Кнопка «Копировать»** копирует короткий URL в буфер обмена.  
- **Кнопка с QR‑кодом** открывает или скачивает QR‑код для этой короткой ссылки (удобно для плакатов, визиток, презентаций).  
- **Кнопка «Поделиться»** позволяет быстро отправить ссылку через доступные способы (поделиться в мессенджерах/соцсетях или скопировать в готовом формате).
